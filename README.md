# PowerBlade

![PowerBlade!](images/PB1.png)

A SwitchBlade application that runs on PowerShell. More to be added as time goes on. Used as a teaching tool. 



Current features:
1. View Services
2. View Processes
3. Empty Recycle Bin


In Progress
